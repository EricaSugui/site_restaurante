Projeto em HTML, CSS.

## Como instalar esse projeto e rodar o sass:
- `git clone` este projeto.
- `cd` este projeto.
- `npm install ` .
- após fazer alterações nos arquivos .scss, rodar o seguinte comando:
- `npm run watch-css`
